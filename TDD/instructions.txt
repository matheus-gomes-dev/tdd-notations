npm test (executar rotina de testes)
npm test -- --bail (pára após encontrar um erro)
npm run test:tdd (fica executando no background, alterando automaticamente após mudanças. Foi incluido --watch no package.json para esse script)
npm run test:coverage (executa nyc (istanbul) para gerar relatório de coverage na pasta coverage. Disponível após executar "npm install --save-dev nyc")
